Caliper

 

caliper is a utility developed for Audio Anecdotes that provides interactive tuning of parameters.  Control is provided by means of keyboard input and does not require a GUI. This facilitates dynamic interaction without complicating simple command line example programs. Basically caliper makes it extremely easy to re-run any command line program tuning and fine-tuning multiple floating point parameters to the program’s command line.

 

caliper based examples are automatically run in a terminal window by clicking on them in Audio Anecdotes’ tour. The following commands may be used to control the program being run by caliper (these single character commands take effect immediately without requiring enter to be pressed):

 

<space>

Run the program with the current parameters              

 

a,b,c,d

Select which parameter to modify

initially a

+,-

Increment or decrement current parameter by current magnitude

 

0-9

Select the magnitude (10^x) to change value by

initially x=1

f

Toggle fractional magnitude (10^-x)

initially not-fractional

z

Zero the currently selected parameter

 

i

Restore the current parameter to initial value

 

!

Toggle immediate re-run of application

initially autorun

?

Print help

 

q

Quit

 

 

 

caliper itself is run from the command line and has the following usage:

 

caliper command [-f list] [-i list]

 

command          string containing the command caliper is to run in printf format with %f controlling where each controllable  parameter should be located (be sure to quote this string on the command line).

 

-i                    optional argument controlling the initial value of each parameter (argument should be followed by a white-space delimited list of floating point values corresponding to the initial value of each variable)

 

-f                    optional argument controlling the default fractional state of the default change magnitude of each parameter (followed by a list of 1 or 0 values to enable or disable fraction mode for each variable)

 

-m          optional argument controlling the default change magnitude of each parameter (followed by a list of integer values between 0 and 9)

 

For example caliper can be used to control a program called sine with –f (frequency in Hz) and –a (attenuation 0-1.0) parameters via the following command:

 

caliper “sine –f %f –a %f” –f 0 1 –i 440 0.8

 

In the above example the frequency parameter defaults to 440Hz with non-fractional increment, and an attenuation of 0.8 with fractional increment. In the default auto re-run mode caliper automatically stops the currently running instance of the program under test and re-runs it with newly updated parameters every time + or – are pressed.
