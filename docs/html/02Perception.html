Perceivable Auditory Latencies - DiFillipo and Greenebaum

These examples demonstrate latency.

 

Run clickfusion which plays a sequence of click pairs, each with decreasing time between the first and second click. Eventually, the clicks cannot be perceived as distinct, but instead blend into a single sound.

Run caliper "clickfusion %f" -i 0.01 to play this demo instrumented with the "caliper" adjustment tool, starting with clicks 0.01 seconds apart.

 

Run sidetone to experiment with how latency affects speech. This demo requires a microphone and headphones. Try running from the command line and varying the latency while speaking or counting.

Play the McGurk_Effect video. In this video, a computer rendered face mouths one sound while a second sound is played. For many people, the conflicting stimulation causes a third sound to be perceived. Try watching the video with the sound disabled or listening to the sound without watching the video.
