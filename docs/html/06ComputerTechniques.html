RampTest: Quantifying Audio Performance - Greenebaum

This example demonstrates RampTest.

The Ramp Test is best experimented with from the command line. Bring a command line shell up then cd to the AudioAnecdotes/Content/06ComputerTechniques/2Greenebaum-Test directory. This demo uses standard I/O to simulate audio because most standard PC's lack digital I/O required for this test. Rampout creates a sawtooth signal that ramptest analyzes for error.
The most simple way to run the test is to pipe the output of rampout to ramptest: rampout | ramptest
Data files with typical errors have been provided for instruction. Those may be run against ramptes by redirection: ramptest < rampdata1
