This tour explores much of the content installed from the Audio Anecdotes CD, however, to get full value from the CD please explore the actual source code, and build and make changes to the code.

This tour is not intended to stand alone but rather to complement the book, #ifdef CD1 Audio Anecdotes Volume 1: Tools, Tips, and Techniques for Digital Audio, #endif #ifdef CD2 Audio Anecdotes Volume 2: Tools, Tips, and Techniques for Digital Audio, #endif #ifdef CD3 Audio Anecdotes Volume 3: Tools, Tips, and Techniques for Digital Audio, #endif #ifdef CD1 ISBN: 1-56881-104-7. #endif #ifdef CD2 ISBN: 1-56881-214-0. #endif #ifdef CD3 ISBN: 1-56881-000-0. #endif Please refer to the book for a detailed explanation of the examples provided here.

The tour will not operate correctly by directly opening the tour.html file, rather it must be served from aathttpd so that the executable demos may be run. tour.bat or tour.sh –w (or –l on Linux, –m on Mac OS X) start aathttpd automatically.

Alternately aathttpd may be run manually from the AudioAnecdotes/Volume1AudioAnecdotes/Volume2AudioAnecdotes/Volume3 directory:

cd AudioAnecdotes/Volume1

#endif #ifdef CD2 style='font-size:10.0pt;font-family:"Courier New"'> AudioAnecdotes/Volume2

#endif #ifdef CD3 style='font-size:10.0pt;font-family:"Courier New"'> AudioAnecdotes/Volume3

#endif

.\bin\aathttpd -p 8085 -D -d . -c "**.cgi"

 

Then use a web browser of your choice to view the tour using the URL:

http://localhost:8085/tour.html

On Windows:

start->run

iexplore http://localhost:8085/tour.html

 

aathttpd will be automatically shut down if the tour is stopped using the red exit link on the tour’s menu bar. Alternately the aathttpd process may be killed manually.

On UNIX (Mac OS X, Linux):

killall aathttpd

 

On Windows:

Press Ctrl-Alt-Del to start the task monitor.

Then highlight the aathttpd process and press “End Process.”
