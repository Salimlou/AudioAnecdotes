Welcome to the Audio Anecdotes Volume 1 tour

This tour explored much of the content installed from the Audio Anecdotes CD however to get full value from the CD please explore the actual source code, build and make changes to the code.

The tour requires thttpd to be running so that demos may be run. tour.bat or tour.sh start thttpd, or thttpd may be run manually from the volume1 directory:

cd AudioAnecdotes/Volume1

.\bin\thttpd -p 8085 -D -d . -c "**.cgi"

 

Use a web browser of your choice with the following url:

iexplore http://localhost:8085/tour.html
