Updating Audio Anecdotes

 

The Audio Anecdotes installer, install.sh, is also useful after installation to automatically check for and install updates posted on the Audio Anecdotes website, http://www.audioanecdotes.com.

 

Windows users may simply run Web Update from the AudioAnecdotesV1 program group:

 

Start->”All Programs”->AudioAnecdotesV1->”Web Update”

 

On UNIX systems (Linux and Mac OS X) or on Windows via Cygwin the installer may be run in update mode manually by starting a text shell, changing directory to the aacd directory in the directory Audio Anecdotes was installed to, then running install.sh with -u (for update mode) and one of the following: -l (Linux) -w (Windows), or -m (Mac OS X). For example, I would do the following if updating my Linux installation on my home directory:

 

cd ~keng/AudioAnecdotes/aacd

sh install.sh –u -l
