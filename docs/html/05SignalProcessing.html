Audio Masking in Audio Compression - Malvar

This example demonstrates audio masking.

 

Run sineMask. The demo repeats the following sequence:

1) Plays a reference 1000 Hz tone for one second.

2) Adds a lower amplitude 1100 Hz tone to the reference (which should be masked).

3) Adds a 1300 Hz tone which doesn't get masked.

 

This cycle repeats (kill the window it creates to stop the demo).

 

Run caliper "sineMask %f %f" -i 1100 0.2 -f 0 1 -m 2 1 to run sineMask in the “caliper” adjustment tool to experiment with what frequencies and amplitudes you can hear and which are masked by the 1000 Hz tone.
