Audio Masking in Audio Compression - Malvar

This example demonstrates audio masking.

Run sineMask. The demo repeats the following sequence: 1) Plays a reference 1000 Hz tone for one second 2) Adds a lower amplitude 1100 Hz tone to the reference (which should be masked) 3) adds a 1300 Hz tone which doesn't get masked.
This cycle repeats (kill the window it creates to stop the demo)
