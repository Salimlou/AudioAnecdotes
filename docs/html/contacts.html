Contact Audio Anecdotes

 

We want to hear from you!  Audio Anecdotes was created as a forum to share tools and techniques with the audio community.  The subjects covered in this volume only scratch the surface of topics we would like to cover. If you have been inspired by Audio Anecdotes, we encourage you to share your own tools, techniques, and experiences!

 

Visit the Audio Anecdotes web site, http://www.audioanecdotes.com, to find the latest errata, download code updates, or find out what is new.

 

Audio Anecdotes was made possible by the hard work and contributions of many people.  If you find an error in the text or code, or have a code improvement, please send it to: errata@audioanecdotes.com.

 

If you would like to contribute to a future volume, please send your name, email address, postal address, and a brief concept to: submission@audioanecdotes.com.

 

Finally, please email comments or suggestions to me at: keng@audioanecdotes.com.
