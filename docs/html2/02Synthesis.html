Band Limited Oscillators using Wave Table Synthesis - Burk

This first example, Play pa_multiwave non-band-limited, demonstrates the problems inherent in a naive sawtooth implementation. Notice the aliasing that becomes obvious as the sawtooth is ramped to higher frequencies. Near the end of the example the aliasing is so bad it dominates the intended signal.

The next example, Play pa_multiwave band-limited, executes the same sawtooth ramp but with bandwidth limiting enabled. The aliasing of the first example should be gone.
